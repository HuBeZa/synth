<H1>GO SYNTH</H1> 
Keyboard operated synthesizer that runs on a simple console

<img src="./img/synth.png" height="600" />

<H2>Features:</H2>
<ul>
    <li>Waveform</li>
    <li>Pan</li>
    <li>Gain</li>
    <li>Automatic Chords</li>
    <li>Overtones</li>
    <li>Tremolo</li>
    <li>ADSR Envelope</li>
    <li>Oscillator</li>
</ul>
